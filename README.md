# Global Media Bias Analyzer

## Project Overview
The **Global Media Bias Analyzer** is a data-driven tool designed to track how different news outlets report on countries around the world.By analyzing sentiment scores from the GDELT Project, this platform identifies whether media coverage is leaning toward a positive, negative, or neutral narrative.

## Key Features
- **Geopolitical Roulette:** Selects a random country to explore new media signals.
- **Bias Classification:** Automatically labels news channels based on sentiment scores.
- **Interactive Analytics:** Displays data through charts and graphs.
- **Source Verification:** Provides direct links to original news articles.

## Technologies Used
* **Python**
* **Streamlit**
* **Pandas**
* **Plotly**
